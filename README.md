# Hotel-V2

Delightful Hotels

## Quick start

Clone the repo:

```
git clone https://github.com/gonzalochuan/Hotel-V2.git
cd Hotel-V2
```

Install dependencies:

```
npm install
```

Run the dev server:

```
npm run dev
```

That's it — the repo excludes `node_modules` via `.gitignore` so cloning and `npm install` will reproduce it locally.
